# 文旅智能体项目 · 学习大纲

## 第一章：开发环境与项目初始化
- **学习方向**：Python环境搭建、依赖管理、项目结构认知、代码规范

## 第二章：配置管理体系
- **学习方向**：YAML配置读取、配置封装设计、多环境配置切换

## 第三章：工具函数层（Utils）
- **学习方向**：日志封装、路径管理、文件处理、Prompt加载器的实现

## 第四章：数据层（Data）
- **学习方向**：本地数据文件结构（JSON/CSV/TXT）、数据读写、数据格式理解

## 第五章：Prompt工程（Prompts）
- **学习方向**：Prompt模板设计、变量注入、输出格式控制

## 第六章：模型层（Model Factory）
- **学习方向**：阿里云百炼API对接、多模型切换、大模型调用封装

## 第七章：RAG向量检索基础
- **学习方向**：Chroma向量库使用、Embedding生成、文档切分、相似度检索

## 第八章：RAG服务封装（RAG Service）
- **学习方向**：RAG流程编排、检索与生成结合、景点导览问答实现

## 第九章：票务客户端层（Client）
- **学习方向**：第三方API对接、网络请求封装、票务数据解析、异常处理

## 第十章：Agent工具层（Tools）
- **学习方向**：自定义工具定义、工具参数校验、中间件设计

## 第十一章：ReAct Agent（React Agent）
- **学习方向**：ReAct推理模式、Agent状态管理、思考-行动-观察循环

## 第十二章：旅游Agent（Travel Agent）
- **学习方向**：业务意图识别、对话状态流转、多轮对话记忆、参数追问

## 第十三章：主入口与项目整合（App）
- **学习方向**：Agent与RAG集成、Flask API封装、项目启动与调试

---

## 整体学习顺序建议

```
第一章 → 第二章 → 第三章 → 第四章 → 第五章 →
第六章 → 第七章 → 第八章 → 第九章 → 第十章 →
第十一章 → 第十二章 → 第十三章
```

**原因**：从基础设施（环境、配置、工具）→ 数据层 → 模型层 → RAG能力 → 业务客户端 → Agent核心 → 项目整合，由底层支撑到上层业务，符合代码依赖顺序。

---

## 项目目录结构说明

| 目录/文件 | 职责 | 归属层级 |
|-----------|------|----------|
| `config/` | 配置文件（YAML） | 基础设施层 |
| `utils/` | 通用工具函数 | 基础设施层 |
| `data/` | 业务数据（景点、订单、规则） | 数据层 |
| `prompts/` | Prompt模板 | 模型/RAG层 |
| `model/` | 大模型工厂 | 模型层 |
| `rag/` | 向量检索与RAG服务 | RAG层 |
| `client/` | 票务API客户端 | 业务客户端层 |
| `agent/` | Agent核心逻辑与工具 | Agent层 |
| `app.py` | 主入口、API暴露 | 应用层 |
