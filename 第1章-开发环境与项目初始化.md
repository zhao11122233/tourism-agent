## 二、核心知识点讲解

### 2.1 项目定位
本项目是**纯C端文旅游客AI智能体**，核心功能：
- 票务优惠判定、票价计算、游览路线规划
- 多轮对话记忆
- 基于 Chroma 的 RAG 景点导览问答
- 自定义工具：ticket_query（查票）、ticket_book（订票）

### 2.2 技术栈总览

| 类别 | 组件 | 用途 |
|------|------|------|
| LLM 框架 | LangChain 0.3.7、LangGraph 0.2.50 | Agent 构建、状态图 |
| 大模型 | 阿里云百炼 qwen-turbo / qwen-plus | 对话生成 |
| 向量库 | Chroma 0.5.15、langchain-chroma | 知识库 RAG |
| 前端 | Streamlit 1.40.1 | Web 可视化界面 |
| 文档处理 | pypdf、langchain-text-splitters | PDF/文本解析 |
| 配置 | pyyaml | YAML 解析 |

### 2.3 架构分层（强制）
```
┌─────────────────────────┐
│  Streamlit UI (app.py)  │  对话界面
├─────────────────────────┤
│  LangGraph Agent        │  意图路由、状态管理、参数校验
├─────────────────────────┤
│  8 Tools                │  工具调用
├─────────────────────────┤
│  TicketClient           │  HTTP 请求、异常兜底
├─────────────────────────┤
│  ChromaDB RAG + Qwen    │  知识检索 + 生成
└─────────────────────────┘
```
**核心约束**：Agent 层严禁直接调用外网 API，必须通过 Client 层。

### 4.2 最小启动入口 app.py
```python
import os

# 禁用Streamlit的使用统计和邮箱提示
os.environ['STREAMLIT_BROWSER_GATHER_USAGE_STATS'] = 'false'

import streamlit as st
from agent.travel_agent import TravelAgent

# 标题
st.title("文旅智能助手 — 查票 · 订票 · 优惠 · 路线 · 导览")
st.divider()

# 侧边栏 — 示例查询
with st.sidebar:
    st.subheader("示例查询")
    st.markdown("""
**查票/订票类：**
- 帮我查一下故宫明天的余票，3个人
- 帮我订八达岭长城后天的票，2个人，手机号13812345678

**票务优惠类：**
- 我有2个老人和1个儿童，想去故宫，有什么优惠政策
- 小孩5岁去黄山要买票吗
""")

if "agent" not in st.session_state:
    # 仅在首次访问时实例化 Agent（Streamlit 会话状态）
    st.session_state["agent"] = TravelAgent()

if "message" not in st.session_state:
    st.session_state["message"] = []

for message in st.session_state["message"]:
    avatar = "🧳" if message["role"] == "user" else "🏯"
    st.chat_message(message["role"], avatar=avatar).write(message["content"])

# 用户输入提示词
prompt = st.chat_input()

if prompt:
    st.chat_message("user", avatar="🧳").write(prompt)
    st.session_state["message"].append({"role": "user", "content": prompt})
```

## 五、验收标准+踩坑避坑点

### 5.1 验收标准
- [ ] Python 3.10+ 已安装
- [ ] `pip install -r requirements.txt` 成功无报错
- [ ] 阿里云百炼 API Key 已配置到环境变量 `DASHSCOPE_API_KEY`
- [ ] 执行 `streamlit run app.py` 后浏览器能打开界面
- [ ] 侧边栏的 4 类示例查询清晰可见

### 5.2 踩坑避坑点
1. **dashscope 版本兼容**：必须使用 `1.20.14`，新版本对 qwen-turbo 接口有调整
2. **Python 版本**：LangGraph 0.2.50 建议 Python 3.10+；3.9 及以下会报语法错误
3. **环境变量配置**：Windows 需用 `setx DASHSCOPE_API_KEY "sk-xxx"`，重启终端生效
4. **Streamlit 端口冲突**：默认 8501，被占用时 `streamlit run app.py --server.port 8502`
5. **第一次启动慢**：Streamlit 首次运行会下载前端资源，需联网
